# Sideline Manager (rollback + mobile polish)

This build rolls the content structure back to the previous working version while keeping the improved mobile positioning:
- restores Quick Swap, Re-enter a player, Game Tools, Rotation Plan, and Previous game records sections
- restores the previous Rotation Plan and Game Tools content/ordering
- removes the solid red treatment from the current block card and returns it to a lighter striped/neutral appearance
- tightens the banner title and tagline so they sit more neatly alongside the icon on mobile
- keeps the more compact left-info / right-buttons timer layout on mobile
